# Changelog

## 1.0.0

- Initial release
- Compact HUD overlay for viewing and adjusting wounds/strain from the canvas
- Support for characters (wounds/strain), minions (wounds only), rivals (wounds only), and vehicles (hull trauma/system strain)
- +1 / -1 adjustment buttons for token owners
- Threshold exceeded warnings
- Permission-aware display (Observer can view, Owner can edit)
- Live updates when actor data changes
- Token controls toolbar toggle button
